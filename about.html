Short History

Early Web

The Web began as a simple document system and evolved into a multimedia platform.

  • 1990s: Static pages
  • 2000s: Interactive applications
  • 2010s+: Semantic HTML and accessibility focus

Learning Outcomes

  1. Understand semantic elements
  2. Embed media with captions
  3. Publish on GitHub Pages

Definitions

Semantic HTML
HTML that describes the meaning of content (e.g., <article>, <nav>).
Accessible
Content that can be used by people with different abilities.
